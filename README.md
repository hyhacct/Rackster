# Rackster

MC 的 AI 服务端，使用 MCP 协议，支持 Minecraft 1.20.1 版本。# mc-ai-mcp


# 用法

加入到 MCP 配置即可

```json
{
  "mcpServers": {
    "minecraft": {
      "command": "npx",
      "args": [
        "-y",
        "github:hyhacct/Rackster",
        "--host",
        "192.168.3.5",
        "--port",
        "3333",
        "--username",
        "Rackster"
      ]
    }
  }
}
```
